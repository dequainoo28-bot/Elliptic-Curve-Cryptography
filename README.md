Elliptic Curve Cryptography
This project is an academic thesis exploring the theory and practical applications of Elliptic Curve Cryptography (ECC) 
a form of public-key encryption widely used in government systems, digital signatures, and blockchain protocols like Bitcoin.

Topics Covered
•	Abstract algebra prerequisites: groups, fields, and finite structures
•Elliptic curve arithmetic (point addition, doubling, scalar multiplication)
•	Cryptographic protocols: Diffie-Hellman key exchange, ElGamal encryption & digital signatures
•	Private key generation using the secp192k1 domain parameters
•	Attacks on discrete logarithm problems (DLP), including Pollard's algorithm
•	Security vulnerabilities: side-channel attacks and twist-security attacks
•	Real-world applications: Bitcoin, smart e-ID systems

Background
Introduced by Koblitz and Miller in 1985, ECC achieves equivalent security to RSA with significantly smaller key sizes, making it ideal for constrained environments.
This thesis investigates its mathematical underpinnings, implementation, and security considerations.
