 \chapter*{Abstract}

Koblitz and Miller introduced the use of elliptic curves in encryption in 1985. The function of elliptic curves in cryptography was investigated.In this thesis we explored the basic implementation and security issues of several elliptic curve cryptosystems.Some of the aspect that will be addressed includes;Using a single solution to obtain many elliptic curve solutions,finding points algebraically,Application of rational numbers on elliptic curves in cryptography and how well programming codes are used to find private key cryptography.
